    Machine Learning, from absolute zero.

    A complete, rigorous, beginner-first path that takes you from "what even is machine learning?" through classical ML, the real machinery of neural networks, and a genuine working understanding of the transformers and large language models behind tools like ChatGPT — plus the practical skill to build things yourself.

    Who this is for

    You. Even if you have never written a line of ML code and the last math you saw was in high school. This course assumes no prior ML knowledge and only basic high-school math — every other piece of math you need is taught gently, right when you need it, intuition first. If you can write a simple for loop in Python (or are willing to learn alongside), you can do this.

    How this course works

    Every module follows the same confidence-building rhythm, refined from how the best courses on earth teach beginners (Andrew Ng, fast.ai, Google, 3Blue1Brown, StatQuest):

    1. Why should I care? — a concrete, real reason the idea matters.
    2. Intuition first — plain language and analogies before any symbols.
    3. Making it precise — the math, introduced gently, every symbol defined, every formula also explained in words.
    4. Worked example by hand — real numbers, every step shown.
    5. In code, twice — first from scratch so the idea is transparent, then the library way (scikit-learn / PyTorch) so you see how little code it takes.
    6. Pitfalls, exercises (with full solutions), and a self-check — so you actually know you know it.

    The order is deliberate: ideas are introduced once, then reused. The cost function and gradient descent you meet in Modules 3–4 power everything up to and including neural networks. Nothing is "left as an exercise"; nothing is hand-waved.

    Your Claude Code study companion

    You're doing this course with a Claude Code instance open on the side — a brilliant move. Treat it as a tireless tutor sitting next to you: ask it to re-explain anything that doesn't click, to quiz you, to run and debug your code, and to pair-program the projects. Every module ends with a green "Work this with your Claude Code companion" box of copy-paste prompts. Module 0 has a full setup guide for using it well — start there.

    What you need

    • A modern web browser (you're using one now).
    • An internet connection while reading — the math and code formatting are rendered live by KaTeX and highlight.js from a CDN.
    • Google Colab to actually run the Python — it's free, needs no installation, and even gives you a free GPU later on. Module 0 walks you through it.

    Your progress is saved automatically

    Tick "Mark module complete" at the top of any module and the progress bars fill in. It's stored in your browser (localStorage), so it survives refreshes and closing the tab. The Reset my progress button at the bottom of the sidebar clears it.
